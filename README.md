# Global Analysis of Attacks on Healthcare Workers in Conflict Zones

This project analyzes global data on attacks against healthcare workers and healthcare facilities in conflict zones using Power BI.

## Project Overview
Wars do not only cause direct casualties but also weaken healthcare systems by targeting hospitals, medical infrastructure, and healthcare workers. This analysis explores patterns in these attacks using publicly available data.

## Dataset
Source: Safeguarding Health in Conflict Coalition (SHCC)

The dataset contains information about:
- Attacks on healthcare workers
- Healthcare worker deaths and injuries
- Types of healthcare professions affected
- Geographic locations of incidents

## Tools Used
- Power BI
- Data Visualization
- Data Analysis

## Key Insights
- 3,623 attacks on healthcare workers and facilities recorded globally
- 932 healthcare workers killed
- 824 healthcare workers injured
- Doctors, nurses, and paramedics are among the most affected professions
- Attacks are concentrated in regions experiencing prolonged armed conflicts

## Dashboard
The Power BI dashboard visualizes:

- Total attacks
- Healthcare worker deaths and injuries
- Attacks by healthcare worker profession
- Global hotspots of attacks

## Files in This Repository
- Healthcare attacks dashboard
- Dataset used for analysis

## Purpose
This project demonstrates the use of data analytics and visualization to understand humanitarian issues affecting healthcare systems during conflicts.
